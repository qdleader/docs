



好多情况下我们在小程序中的写的计时器清除不掉，很多时候在方法内部定义的计时器，在外面清除怎么也清不掉（虽然有变量提升


这时候在小程序最外面  定义

```
var timerLeft = null;




countTimeDownNew: function (manager, cancel) {
  let that = this;
    timerLeft =  setInterval(()=> {
         let num = this.data.num3 + 1
         this.setData({
           num3:num
         })
     },1000)
},


playOrpause: function () {
  clearInterval(timerLeft)
},
```


即可手动开启，关闭计时器，挺好用的
