

## 方法1
```js
<Form
  {...formItemLayout}
  form={form}
  >
  <Form.Item name="username" label={"用户名"} >
    <Input />
  </Form.Item>
  
  // 中间加一个类型为 password、隐藏的输入框
  <Form.Item style={{ width: 0, height: 0, padding: 0, margin: 0 }}>
    <Input name="password" style={{ visibility: 'hidden', height: 0 }} />
  </Form.Item>
  
  <Form.Item  name="password" label="密码">
    <Input type="password" />
  </Form.Item>
</Form>
```
