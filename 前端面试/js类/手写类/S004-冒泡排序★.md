
# 手写冒泡排序

```
function sort(elements) {
  for(let i = 0;i < elements.length; i ++) {
    for(let j = 0; j < elements.length - i - j; j ++) {
      if(elements[j] > elements[j + 1]) {
        let swap = elements[j];
        elements[j] = elements[j + 1];
        elements[j + 1] = swap
      }
    }
  }
}
```
