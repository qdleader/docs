等边三角形呢？

对于等边三角形我们知道，高是边的sqrt(3)/2倍，所以如果高为40的话，两边就是差不多23

```
.triangle {
				border-left: 23px solid transparent;  
				border-right: 23px solid transparent;  
				border-bottom: 40px solid red;  
				width: 0px;
				height: 0px;
			}
```
