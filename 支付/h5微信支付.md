
# h5微信支付

微信内支付用sdk，这在微信公众开发的文章已详细提到。
我们在这说一下，微信外浏览器微信支付。

我们在这先判断是微信内浏览器还是微信外浏览器。

```js

      // #ifdef H5
      const ua = window.navigator.userAgent.toLowerCase();
      if (ua.match(/micromessenger/i) == 'micromessenger') {
          return true;
      } else {
          return false;
      }
      // #endif
  },
  ```
