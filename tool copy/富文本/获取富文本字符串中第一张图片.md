js获取富文本中的第一张图片url正则公式及去除字符串里面的html标签
后台发来一个富文本字符串里面可能包含了0、1、2、3…个图片标签（img）,我们的任务是获取这个字符串里面第一张图片的url，如果没有图片则返回空

```js
var imgUrlFun = function(str){
        var data = '';
            str.replace(/<img [^>]*src=['"]([^'"]+)[^>]*>/, function (match, capture) {
                  data =  capture;
            });
        return data
    }

```
