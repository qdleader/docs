## 单个作用域插槽的简写
 限定范围的插槽非常有趣，但为了使用它们，您还必须使用许多模板标记。幸运的是，有一个简写可以让我们摆脱它，但只有在我们使用单个作用域槽的情况下。

你或许这样使用过
```
<TableList>
  <template #header="attrList">
    <TableHeader v-bind="attrList" />
  </template>
</TableList>
 // 不使用 template

<TableList #header="attrList">
  <TableHeader v-bind="attrList" />
</TableList>
```
